Source code : First of all, we suggest you to create your own Makefile, which should automatically compile your sources and generate the executable that you can send to the robot in the robot using an SSH connection. You may find an example of Makefile in our Gitlab repository See Gitlab here.

Easy to use: compile the code

In order to download it enter the path where you would like to place the folder containing the code and run: 

git clone git@gitlab.eurecom.fr:monniot/OS_Rover_APA.git

Then you may either download the whole source code inside the robot and compile it internally or use docker on your machine in order to cross compile it and then download the code inside the robot. First of all make sure that your computer is connected to the robot via bluetooth connection. Then, if you want to compile the source code internally, from the chosen installation path type the following commands on the terminal: 
cd OS_Rover_APA

make docker

It will open the compiler: 
docker run --rm -it -h ev3 -v ~/OS_Rover_APA/:/src -w /src ev3cc /bin/bash

make install blue

make

In a new terminal :
scp main robot@ADRESS:

Check the main is sent to the robot!
