Description of the algorithms used in the project

Posted for OS Project Fall 2018

This section contains a description of the main algorithms we used. The functions linked to these algorithms are also described, using a brief summary.
Before delving into each part notice that the division into topics was done only to enhance understandability. In fact, each of the following sections is strictly connected to all the other ones. The structure of the code is divided in three parts: one for bluetooth connection using a thread, one for the main function of the program, the last one for the use of the motors to complete the actions

Main functions

Concerning the functions used, there are 3 main functions called in the main, the first one called 

initial throw()
enables the catapult in order to throw the 2 balls from the beginning.
The second and the third one : 
 search_ball_right()
and 
 search_ball_left()
describes how the robot searches for the ball.


initialthrow() {
            throw () 
            turn motor left () 
            throw ()  
          
}


Then we have two functions for searching the ball. We divide the space in two parts :left and right. The following pseudo code explain the principle for the right side(which could be easily reproduce with the left side):


search_ball_right() {
          while (angle <90 degres){
              searchball(treshold) 
              if found: 
              flagdetected=1
              break
              throw() 
              }
              
          
           if flag detected{
                        turn (180 degres)
                        getball( )
                        turn (180 degres)
           }       
           turn (initial position)
           if flag detected and flag_ball_caught{
                              throw()
           }                  
          }


To get more details in the previous main functions used, here is described the action functions used for picking up the balls

getball () {
            disable_catapult () 
            move_forward () 
            enable_catapult () 
            move_backward ()
          
}
